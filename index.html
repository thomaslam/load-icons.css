load-icons.css

A collection of free pure CSS3 load icons to use for your projects.

1. Download via Bower 

$ bower install load-icons-css

or clone git repo

$ git clone git@github.com:thomaslam/load-icons.css.git

2. Include appropriate link tag, e.g. 

<link rel="stylesheet" href="./bower_components/load-icons-css/css/load-icons.css">

3. Include the appropriate div to where you want the load icon to appear
